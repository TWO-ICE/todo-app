## 一、任务目标

将现有的 Todo App 从"工具型应用"升级为"让人想用下去"的游戏化体验应用，通过 RPG 元素、奖励机制和收集系统提升用户粘性和使用乐趣。

**核心理念**: 每完成一个任务都是一次冒险，每一天的使用都是一场游戏！

---

## 二、核心功能（按优先级顺序）

### 🎯 第一优先级：副本系统 + XP奖励

#### 功能描述
将完成任务设计成"攻打副本"的游戏体验，不同难度的任务对应不同等级的副本。

#### 实施要求

**1. 副本难度分级**
- **N级副本**（Normal）
  - 对应：Low 优先级任务
  - XP 奖励：+10 XP
  - 视觉标识：绿色徽章，简朴边框
  - 完成特效：绿色粒子 + "N级副本通关"文字

- **R级副本**（Rare）
  - 对应：Medium 优先级任务
  - XP 奖励：+25 XP
  - 视觉标识：蓝色徽章，银色边框
  - 完成特效：蓝色闪光 + "R级副本通关"文字

- **SR级副本**（Super Rare）
  - 对应：High 优先级任务
  - XP 奖励：+50 XP
  - 视觉标识：紫色徽章，金色边框
  - 完成特效：紫色爆炸 + "SR级副本通关"文字

- **SSR级副本**（Super Super Rare）
  - 对应：连续完成3个以上任务时触发
  - XP 奖励：+100 XP
  - 视觉标识：彩虹色徽章，动态光效
  - 完成特效：全屏彩带 + "SSR级副本通关！" + 特殊音效提示

**2. XP 奖励机制**
- 总 XP 显示在界面顶部，醒目位置
- 每次完成任务，XP 实时增加并有数字滚动动画
- XP 累计进度条，显示当前等级进度
- 连续完成任务触发连击加成（2连击+20%，3连击+30%，最高50%）

**3. 视觉效果**
- 副本卡片带有对应稀有度的边框和背景
- 完成时播放对应级别的动画特效
- 使用 FontAwesome 或 Lucide 图标显示稀有度符号（N/R/SR/SSR）

---

### 🧩 第二优先级：拼图收集系统

#### 功能描述
完成任务有几率掉落"拼图碎片"，收集碎片可解锁完整的拼图成就。

#### 实施要求

**1. 碎片掉落机制**
- 每完成一个任务，随机掉落 1-3 个碎片
- 掉落率：100%（至少1个）
- SSR 级副本必掉 2-3 个稀有碎片

**2. 拼图主题**
设计 3-5 套拼图主题，每套包含 9 片碎片（3x3）：
- 🌸 春日樱花（绿色系）
- 🌊 夏日海浪（蓝色系）
- 🍁 秋日枫叶（红色系）
- ❄️ 冬日雪花（白色系）
- 🌈 彩虹梦境（彩色系）

**3. 收集册界面**
- 新增"图鉴"标签页
- 显示所有拼图主题和收集进度
- 已收集的碎片显示彩色，未收集显示灰色剪影
- 每完成一套拼图，解锁特殊徽章和奖励（+200 XP）

**4. 数据持久化**
- 使用 localStorage 保存碎片收集状态
- 包含：碎片ID、所属主题、获得时间

### 🎨 第四优先级：UI/UX 优化

#### 功能描述
全面升级视觉设计，营造沉浸式 RPG 游戏氛围。

#### 实施要求

**1. RPG 主题设计**
- **配色方案**：深色主题为主，金色点缀
  - 主色：#1a1a2e（深蓝黑）
  - 辅色：#16213e（午夜蓝）
  - 强调色：#f4a261（金橙色）

  - SSR色：#9b59b6（紫色）、#f39c12（金色）、#e74c3c（红色）渐变

- **字体**：
  - 标题：Display 字体（如 'Press Start 2P' 或 'Cinzel'）
  - 正文：无衬线字体（Inter 或 Roboto）
  - 数字：等宽字体（显示XP和等级）

**2. 动画效果**
- **任务完成动画**：
  - 数字滚动（+10 XP → +25 XP → ...）
  - 粒子爆炸效果
  - 徽章闪烁
  - "副本通关"文字弹出

- **升级动画**：
  - 屏幕闪烁金光
  - 等级数字从旧值滚动到新值
  - 新徽章从天而降
  - 音效提示（可选）

- **碎片掉落动画**：
  - 碎片从任务卡片飞出
  - 旋转缩放入库
  - 图鉴标签页红点提示

**3. 交互优化**
- Hover 状态：卡片上浮 + 发光
- 点击反馈：波纹效果
- 加载状态：像素风格加载动画
- 空状态：可爱的像素角色提示

**4. 沉浸式元素**
- 顶部：玩家信息栏（头像 + 等级 + 称号 + XP条）
- 左侧：任务列表（副本列表）
- 右侧：图鉴入口 + 统计数据
- 底部：功能导航（任务/图鉴/成就）

## 三、技术实施建议

### 技术栈
- **前端框架**：React 19.2.4（已安装）
- **状态管理**：React Context + Hooks（useTodos 扩展）
- **动画库**：Framer Motion（推荐）或 CSS Animations
- **图标库**：Lucide React（已安装）+ Font Awesome（可选）
- **数据持久化**：LocalStorage + JSON Schema

### 数据结构设计

```typescript
// 用户游戏数据
interface GameData {
  xp: number;
  level: number;
  title: string;
  badges: string[];
  achievements: string[];
  fragments: Fragment[];
  puzzleProgress: PuzzleProgress[];
  comboCount: number;
  lastLoginDate: string;
}

// 碎片数据
interface Fragment {
  id: string;
  theme: string;
  index: number; // 1-9
  obtainedAt: string;
  rarity: 'common' | 'rare' | 'epic';
}

// 拼图进度
interface PuzzleProgress {
  theme: string;
  collected: number[]; // 已收集的碎片索引
  completed: boolean;
  completedAt?: string;
}

// 任务扩展
interface TodoItem {
  // ... 原有字段
  rarity: 'N' | 'R' | 'SR' | 'SSR';
  xpReward: number;
}
```
